¿De que trata esta pagina?

Esta pagina sirve para mostrar fotos de determinado dia mediante una Api de la NASA segun lo indique el usuario.

¿Qué es la API de la NASA?

La API de la NASA es una interfaz de programación que permite a los desarrolladores acceder fácilmente a datos e imágenes relacionados con el espacio, el cosmos y la exploración espacial que la NASA pone a disposición públicamente.
